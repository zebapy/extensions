# Changelog

## [1.0.0] - 2025-12-01

### Added
- Send quick emails to yourself via SMTP directly from Raycast
- Simple form interface for composing subject and body
- Support for multiple recipients with automatic dropdown when configured
- Configurable SMTP settings (host, port, encryption mode)
- Support for SSL/TLS, STARTTLS, and unencrypted connections
- Optional authentication with username and password
- Self-signed certificate support for trusted self-hosted servers
- Customizable sender name and email address
- Named recipient format support (e.g., "Jane Smith <jane@example.com>")
- Credentials stored securely in Raycast's encrypted preferences
