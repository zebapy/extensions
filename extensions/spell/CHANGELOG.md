## [Windows Support and Various Improvements] - 2025-11-06

- Add windows support
- Add action to paste selected word into active application (`⇧` + `↵`)
- Add search caching - Previously searched terms now load instantly and are available offline, making the extension feel much snappier
- Add support for Spanish, accessible via command preferences
- Search results now respect the casing of your query. 
    - For example, 'HELLO' will return results in all caps, and 'Hello' will return title-cased results.
- Fix bug that prevented the results list from clearing when search bar was empty
