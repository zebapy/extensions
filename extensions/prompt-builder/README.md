# Prompt Builder

**Prompt Builder** is a Raycast extension that helps you craft the perfect AI prompts efficiently. It makes it easy to define roles, tasks, tone, audience, and constraints, helping make AI output more accurate and consistent. Build structured prompts with templates, preview them, and copy instantly to your clipboard.

### Features

-  Build structured prompts with role, task, tone, audience, constraints, reasoning, examples, and more
- Save, update, and reuse templates
- Adjustable creativity level and answer tone
- Preview prompts before copying
- Copy prompt to clipboard with one click
- Clear form to start fresh

### Usage

1. Open Raycast and search for **Prompt Builder**
2. Fill in the **form** fields
3. **Save** it as a template if you want to reuse it
4. **Preview** your prompt to check it before copying
5. **Copy** your prompt to clipboard

### Working with Templates

- **Save**: Press `⌘ + S` / `Ctrl + S` to save current prompt as a template
- **Use**: Select a template from the dropdown to auto-fill the form
- **Update**: Press `⌘ + U` / `Ctrl + U` to update the selected template
- **Delete**: `⌘ + ⇧ + D` / `Ctrl + Shift + D` current template.

### Keyboard Shortcuts

- `⌘ + ↵` / `Ctrl + Enter`: Copy to clipboard
- `⌘ + D` / `Ctrl + D`: Clear form
- `⌘ + Y` / `Ctrl + Y`: Preview prompt
- `⌘ + E` / `Ctrl + E`: Edit preview
- `⌘ + S` / `Ctrl + S`: Save as template
- `⌘ + U` / `Ctrl + U`: Update current template
- `⌘ + N` / `Ctrl + N`: New empty template
- `⌘ + ⇧ + D` / `Ctrl + Shift + D`: Delete current template
- `⌘ + ⇧ + ⌥ + D` / `Ctrl + Shift + Alt + D`: Delete all templates
