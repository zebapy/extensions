# Meteoblue Lookup

Check the weather on Meteoblue directly from Raycast.

## Setup

This extension requires a Meteoblue API key to function.

1.  Obtain an API key from [Meteoblue](https://content.meteoblue.com/en/access-data/weather-api).
2.  Install the extension.
3.  Open Raycast preferences (`Cmd + ,`) -> Extensions -> Meteoblue Lookup.
4.  Enter your API Key in the preferences.

## Commands

-   **Meteoblue Weather**: View current weather conditions and forecast.
-   **Today's Forecast**: Detailed hourly forecast for today.
-   **5-Day Forecast**: Daily forecast for the next 5 days.
-   **Will It Rain?**: Quick check for rain probability and amount for the rest of the day.

## Preferences

-   **API Key**: Your Meteoblue API key (Required).
-   **Temperature Unit**: Fahrenheit (°F) or Celsius (°C).
-   **Wind Speed Unit**: mph, km/h, m/s, or kn.
-   **Precipitation Unit**: Inches (in) or Millimeters (mm).
