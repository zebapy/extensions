# Changelog

## [1.0.0] - 2025-11-28

### Added

- Initial release of Unify Path Separator extension
- Convert path separators in clipboard content from forward slashes to backslashes
- Convert path separators in clipboard content from backslashes to forward slashes
- List view with two conversion options:
  - Convert to backslash (replace forward slashes with backslashes)
  - Convert to forward slash (replace backslashes with forward slashes)
- HUD notifications for successful conversions and error handling
- Empty clipboard detection with user feedback
