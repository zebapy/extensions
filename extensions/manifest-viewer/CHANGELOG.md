## [1.0.0] - 2025-11-24

### Added
- Initial release of Manifest Viewer extension
- Support for HLS (HTTP Live Streaming) manifest files (.m3u8)
- Support for WebVTT subtitle files (.vtt)
- Interactive navigation to child manifests and variant playlists
