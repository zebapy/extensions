## [1.0.0] - 2025-11-14

### Added

- Initial release of WebBlocker
- **Add Website to Block** command - Add any website to your block list with automatic URL sanitization
- **Enable Website Blocking** command - Activate system-level blocking for all enabled websites with Touch ID authentication
- **Disable Website Blocking** command - Deactivate blocking while preserving your block list
- **Manage Blocked Sites** command - View, enable/disable, or delete websites from your list
- **Force Re-Block & Fix** command - Troubleshoot and refresh blocking if issues occur
- System-level blocking that works across all browsers (Chrome, Safari, Arc, Firefox, Edge, Brave)
- Automatic tab closing for already-open blocked websites
- Touch ID authentication for all blocking operations
- Persistent blocking that survives system restarts
- Individual website enable/disable controls
- Smart URL sanitization (handles www, https, paths automatically)
- 100% effective blocking with no bypass options
- MIT License
