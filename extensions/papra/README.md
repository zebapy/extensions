# Papra

This is a Raycast extension for [Papra](https://papra.app/) - _Simple & Secure Document Archiving for Long-Term Storage_.

## ðŸš€ Getting Started

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/papra) OR `install` via Raycast Store

    <a title="Install papra Raycast Extension" href="https://www.raycast.com/xmok/papra"><img src="https://www.raycast.com/xmok/papra/install_button@2x.png?v=1.1" height="64" alt="" style="height: 64px;"></a>

2. **Get your Papra URL**:

    - If you are using Papra from the website, use the default URL i.e. `https://dashboard.papra.app/`
    - If you are self-hosting, enter the fully-qualified URL e.g. `https://papra.example.com`

3. **Get your Papra API Token**:

    `Follow` [instructions](https://docs.papra.app/resources/api-endpoints/#authentication)
