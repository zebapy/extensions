# PlanWell Changelog

## [1.1.0] - 2025-12-04

### Changed
- Updated vault schema to use modern format (`occurrence 1` with space, not hyphen)
- New classes now include `startDate`, `date`, `periodId`, and `created` timestamp
- Integrated shared design system tokens for consistent UI colors

### Fixed
- Class creation now follows unified vault schema for cross-app compatibility

## [Initial Version] - 2025-11-27

- Today's Classes: View your schedule with period times and remaining time
- Upcoming Events: See multi-day events with start/end dates
- Todos: View and mark todos as complete
- Quick Add: Add classes to free periods
- Delete Class: Move classes to bin
- Open in PlanWell or external editor (Obsidian, VS Code, etc.)
