# PlanWell for Raycast

A Raycast extension for [PlanWell](https://planwellmd.com) - the markdown-native teacher planning app.

## Features

### Today's Classes
- View your schedule for today with period times
- See remaining time until next class
- Quick add classes to free periods
- View and navigate class notes
- Delete classes (moves to bin)

### Upcoming Events
- View multi-day events
- See time until start/end
- View event notes

### Todos
- View pending and completed todos
- Mark todos as done/pending
- View todo notes

### Open in External Editor
- Open class/event notes in PlanWell or Obsidian
- Configurable external editor (VS Code, TextEdit, etc.)

## Settings

- **Vault Path**: Path to your PlanWell vault folder
- **Default View**: Choose between "All Periods" or "Classes Only"
- **External Editor**: App to open markdown files
- **Obsidian Vault Name**: Required if using Obsidian as external editor
- **Obsidian Subfolder**: For symlink setups (optional)

## Keyboard Shortcuts

- `Enter` - Primary action (Show Notes / Quick Add)
- `Cmd+O` - Open in PlanWell
- `Cmd+Shift+O` - Open in External Editor
- `Cmd+Backspace` - Delete class
- `Cmd+R` - Refresh list

## Requirements

- [PlanWell desktop app](https://planwellmd.com) installed
- A PlanWell vault folder
