# Changelog

## [1.0.0] - 2025-11-07

### Added

- Initial release of AppGrid
- Grid view for macOS applications
- Smart category grouping (Development, Productivity, Communication, Entertainment, Utilities, Others)
- Frequently used apps tracking and categorization
- Multiple view modes (All apps by name, All apps by usage, Category groups)
- Usage statistics tracking
- Multi-language support (English, Korean, Japanese)
- Quick search functionality
- Actions: Launch app, Show in Finder, Copy path, Copy bundle ID
- Custom orange grid icon
- Auto-detection of system language
