# iShader

Apply artistic shader effects to images directly from Raycast.

## Features

- **Dither Effects**: Multiple dithering patterns with customizable parameters
- **ASCII Art**: Convert images to ASCII art
- **Brick Effect**: Create LEGO-like brick patterns
- **Real-time Preview**: See changes instantly with zoom controls
- **Clipboard Support**: Paste images directly from clipboard (Windows)

## Usage

1. Select an image file or paste from clipboard
2. Choose an effect from the effects menu
3. Adjust parameters using keyboard shortcuts or settings form
4. Save the processed image

## Keyboard Shortcuts

- Use number keys and letters for quick parameter adjustment
- See metadata panel for all available shortcuts
