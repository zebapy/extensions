# Cyberduck

This is a Raycast extension for [Cyberduck](https://cyberduck.io/) - _Libre server and cloud storage browser for Mac and Windows with support for FTP, SFTP, WebDAV, Amazon S3, OpenStack Swift, Backblaze B2, Microsoft Azure & OneDrive, Google Drive and Dropbox_.

> NOTE: this extension requires the application to be installed.
