# AI by Vercel

A Raycast extension that integrates the capabilities of Vercel AI Gateway directly into Raycast.

## Features

- **Ask Vercel AI**: Query AI models directly through Vercel's AI Gateway
- **Compare Models**: Ask the same question to multiple AI models and compare their responses
- **Conversation Management**: Save and manage your AI conversations
- **Flexible Model Selection**: Choose from any AI model supported by Vercel AI Gateway

## Setup

1. Install the extension from Raycast Store
2. Login to your Vercel account and obtain your API key from the `AI Gateway` settings page.
3. Configure your preferred AI model(s) in extension preferences

## Configuration

- **AI Gateway API Key** (required): Your Vercel AI Gateway API key
- **AI Model** (optional): Specify one or multiple models separated by commas (default: `openai/gpt-oss-120b`)

Browse available models at: https://vercel.com/ai-gateway/models
