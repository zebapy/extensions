# Jitsi Changelog

## [Added Preferences and Enhanced Functionality] - 2025-11-25

- Added custom Jitsi server support - use your own server instead of meet.jit.si
- Added multiple meeting name format options (timestamp, random words, custom prefix, random only)
- Added default username setting - automatically fill in your name when joining meetings
- Added auto-open browser preference - choose whether to automatically open meetings
- Added start with video muted option
- Added start with audio muted option
- Improved URL generation - clean shareable links copied to clipboard, personalized links open in browser
