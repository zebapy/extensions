# APIs Guru Search

Search [APIs.guru](https://apis.guru/), the largest public directory of machine-readable API definitions.

## Features

- Search through thousands of public APIs by name, provider, or category
- Browse APIs by category (cloud, financial, social, messaging, etc.)
- View API details including version, description, license, and contact info
- Copy Swagger/OpenAPI specification URLs to clipboard
- Open documentation and specifications directly in your browser

## Usage

1. Open Raycast and type "Search APIs" or invoke the command directly
2. Start typing to search for APIs by name or provider
3. Use the category filter dropdown to narrow results
4. Select an API to view details or use available actions

## Actions

- **Open Documentation** - Open the API's main documentation link
- **Open Swagger URL** - View the OpenAPI/Swagger specification
- **Copy Swagger URL** - Copy the specification URL to clipboard
- **Copy Contact Email** - Copy the API maintainer's email (if available)
