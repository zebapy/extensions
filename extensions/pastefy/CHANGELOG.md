# Pastefy Changelog

## [Windows Support + 2 New Create Preferences] - 2025-11-05

- Add Windows Support
- "Delete Paste" `Action` now has a `shortcut` and destructive `style`
- "List Pastes" caches results
- Deletion is optimistic and shows feedback (`Toast`)
- In "Create":
    1. can now choose to remember last selected visibility (ref: [Issue #22641](https://github.com/raycast/extensions/issues/22641))
    2. form now resets on create
    3. can specify one of 4 actions to be done after creation - "Copy & Open in Browser", "Open in Browser Only", "Copy to Clipboard Only", "Do Nothing" (ref: [Issue #22639](https://github.com/raycast/extensions/issues/22639))

## [Initial Version] - 2024-09-17

- Initial release with paste creation and list.
