# Pastefy

Use [Pastefy](https://pastefy.app/) in **Raycast** to _create_, _list_ and _delete_ pastes.

## Configuration

You need to create an API-Key at https://pastefy.app/apikeys if you want to save pastes to your account or show your created pastes in a list.

### Paste to Pastefy

- **Remember Visibility** - check this if you want the extension to remember last selected visibility
- **After Create** - specify what should happen after creation: "Copy & Open in Browser", "Open in Browser Only", "Copy to Clipboard Only", "Do Nothing"
